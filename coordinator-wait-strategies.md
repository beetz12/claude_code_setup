# コーディネーター待機戦略

## 基本待機パターン

### 1. タスク複雑度別の待機戦略

#### 簡単なタスク（複雑度 1-3）
```
初回チェック → 1分待機 → チェック → 2分待機 → チェック → ... → 10分待機（最大）
N = 1, 2, 3, ..., MAX(10)
```

#### 中程度のタスク（複雑度 4-7）
```
初回チェック → 2分待機 → チェック → 3分待機 → チェック → ... → 15分待機（最大）
N = 2, 3, 4, ..., MAX(15)
```

#### 複雑なタスク（複雑度 8-10）
```
初回チェック → 5分待機 → チェック → 5分待機 → チェック（固定間隔）
総実行時間: 最大30分
```
